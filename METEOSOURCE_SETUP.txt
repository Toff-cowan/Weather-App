The app now uses Meteosource API for real-time hurricane and weather data!

STEP 1: Get Your FREE Meteosource API Key
------------------------------------------
1. Visit: https://www.meteosource.com/
2. Click "Sign Up" or "Get Started"
3. Create a free account
4. Go to your dashboard and copy your API key

STEP 2: Add the API Key to Your .env File
------------------------------------------
Open your .env file and add this line:

METEOSOURCE_API_KEY=your_actual_api_key_here

Replace "your_actual_api_key_here" with the key you copied from Meteosource.

Your complete .env file should look like:

GOOGLE_GEMINI_KEY=your_gemini_api_key
DATABASE_URL=mongodb://localhost:27017/
SENDGRID_API_KEY=your_sendgrid_api_key
EMAIL_FROM=your_email@example.com
EMERGENCY_EMAIL_TO=emergency_email@example.com
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE_NUMBER=
EMERGENCY_PHONE_NUMBER=
METEOSOURCE_API_KEY=your_actual_api_key_here
PORT=5000
NODE_ENV=development

STEP 3: Restart the Backend
----------------------------
After adding the API key, restart your backend server:
npm run server
